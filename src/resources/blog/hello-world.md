Welcome to my blog! This is where I'll be writing about things I find interesting — hardware, software, embedded systems, and the occasional deep-dive into whatever project has consumed my evenings lately.

## Why a blog?

I've always kept notes while working on projects. Scattered markdown files, README drafts that never got published, half-finished write-ups. This is an attempt to actually share some of that.

The goal isn't to write polished tutorials — more like a technical diary. Things that worked, things that didn't, and the reasoning behind decisions.

## What to expect

Mostly project post-mortems and exploratory write-ups:

- **Embedded systems** — FPGA, microcontrollers, real-time stuff
- **Software tools** — things I've built or found useful
- **Experiments** — neural networks trained on shoestring resources, generative art, etc.
