Porypal is a specialized image processing tool designed for Pokémon Gen 3 ROM hacking and decompilation projects (pokeemerald/pokefirered). It automates sprite and tileset conversion while maintaining strict adherence to the Pokémon Gen 3's 16-color palette specifications.\n\n\n## Key Features\n\n  - Automatically convert every pixel from an input image to its closest color in a JASC-PAL file.\n  - Multi-palette preview interface for comparison and cherry-picking.\n  - Prioritizes conversions that maintain the highest number of distinct colors (up to 16)\n  - Configurable tileset transformation pipeline via YAML\n  - Interactive tileset editor for precise tile arrangement and manipulation\n  - Automation tools for batch processing and preset application\n  - Zoom and pan controls for detailed tile inspection\n  - Save and load layout presets for consistent tile arrangements\n\n![Porypal UI](docs/img/main_ui.png)\n*Palette editor GUI for palette conversion*\n\n![Tileset UI](docs/img/tileset_editor_ui.png)\n*Tileset conversion interface for the tilesheet modification*\n\n![Automation UI](docs/img/automation_ui.png)\n*Automation interface to apply the same layout to an entire folder*\n\n\n## Installation\n\n### Prerequisites\nEnsure you have Python 3.6+ installed on your system. You can download Python from the [official website](https://www.python.org/downloads/).\n\n### 1. Clone the repository or download the code\nDownload the repository to your local machine or clone it using Git:\n\n\n```bash\ngit clone https://github.com/Loxed/porypal.git\n```\n**Note**: _I recommend cloning the project inside your pokefirered/pokeemerald's `tools` folder._\n\n### 2. Install dependencies\n\n#### Windows\n1. Open Command Prompt (`cmd`) or PowerShell.\n2. Navigate to the project directory:\n\n   ```bash\n   cd \path\to\decomp\tools\porypal\n   ```\n\n3. Install the required dependencies using `pip`:\n\n   ```bash\n   pip install -r requirements.txt\n   ```\n\n#### macOS / Linux\n1. Open Terminal.\n2. Navigate to the project directory:\n\n   ```bash\n   cd /path/to/decomp/tools/porypal\n   ```\n\n3. Run the installer:\n    ```bash\n    ./setup.sh\n    ```\n\n### 3. Run the application\n\nAfter installing the dependencies, you can run the `main.py` script.\n\n   ```bash\n   python3 main.py\n   ```\n\n### 4. Build from source\n\nTo build the program, use the following commands:\n\n\n-  Build the executable with PyInstaller\n```bash\npip install pyinstaller\npyinstaller main.spec\n```\n\n- Copy additional resources to the dist folder\n```bash\ncp config.yaml dist/config.yaml\ncp -r ressources dist/ressources\ncp -r palettes dist/palettes\ncp -r docs dist/docs\ncp -r example dist/example\n```\n\n## Directory Structure\n```\nporypal/\n├── controller/        # Controller components (MVC pattern)\n├── model/            # Model components (MVC pattern)\n├── view/             # View components (MVC pattern)\n├── ressources/       # Application resources\n│   └── porypal.ico   # Application icon\n├── presets/          # Preset configurations\n│   └── ow_sprite.json # Overworld sprite preset\n├── example/          # Example files and reference images\n├── palettes/         # JASC-PAL color definitions\n├── docs/             # Documentation and guides\n│   └── img/          # Documentation images\n├── build/            # Build output directory\n├── .venv/            # Python virtual environment\n├── config.yaml       # Pipeline configuration\n├── main.py           # Core application logic\n├── main.spec         # PyInstaller specification file\n├── class.puml        # UML class diagram\n├── requirements.txt  # Python dependencies\n├── setup.sh          # Unix installation script\n└── LICENSE           # MIT License file\n```\n\n## License\nThis project is licensed under the MIT License - see the [LICENSE](LICENSE) file for details.\n\n## Contact\nFor questions or support, reach out to `prison_lox` on Discord.\n\n## Credits\n\nThe examples used in the documentation come from:\n- [Gen 5 Characters in Gen 4 OW style 2.0](https://web.archive.org/web/20231001155146/https://reliccastle.com/resources/370/) by DiegoWT and UltimoSpriter\n- [ALL Official Gen 4 Overworld Sprites v1.5](https://eeveeexpo.com/resources/404/) by VanillaSunshine\n
